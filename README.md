# BRTS Portal - Bus Rapid Transit System Management Web App

A full-stack web application developed for managing the Bus Rapid Transit System (BRTS).  
The system provides functionalities for passengers, administrators, and transport management.

---

## 🚀 Features

- User Login & Registration
- Bus Route Management
- Ticket Booking System
- Passenger Management
- Admin Dashboard
- Real-Time Transport Information
- Secure Database Handling

---

## 🛠 Technologies Used

- Python
- Flask / Django
- HTML
- CSS
- JavaScript
- SQLite / MySQL
